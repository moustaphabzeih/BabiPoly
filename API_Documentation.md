# BabiPoly API Documentation

## Quick Start
**Base URL:** http://localhost:8080

## Authentication Endpoints

### 1. Register User
- **URL:** POST /api/register
- **Data:** name, email, password, password_confirmation

### 2. Login User  
- **URL:** POST /api/login
- **Data:** email, password
